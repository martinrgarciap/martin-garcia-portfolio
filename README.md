# Martin Garcia Portfolio

A personal portfolio website built to showcase my technical skills, projects, and experience as a software developer. The site is designed to present my work through a clean, modern interface and support my professional brand online.

## Live Demo

https://martin-garcia-website-23ad60d8ee27.herokuapp.com/

## Overview

This portfolio serves as a central place to present who I am, what I build, and the technologies I work with. It is structured to introduce my background, highlight my skills, and create a polished professional presence online.

The homepage currently includes:

- an introduction section
- a skills section
- a modern responsive UI

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Material UI
- Emotion
- Headless UI
- Heroicons

## Features

- Responsive portfolio homepage
- Modern hero section with gradient styling
- Animated role and title text
- Skills and technologies section
- Component-based structure for scalability
- Built with a combination of Tailwind CSS and Material UI

## Scripts

Run the project locally with:

npm install
npm run dev

Build for production:

npm run build
npm run start

Run linting:

npm run lint

## Main Libraries Used

- next for the application framework
- react and react-dom for UI development
- typescript for type safety
- tailwindcss for utility-first styling
- @mui/material for UI components
- @emotion/react and @emotion/styled for styling support with MUI
- react-type-animation for animated text effects
- @headlessui/react and @heroicons/react for accessible UI patterns and icons

## Purpose

I built this portfolio to create a stronger online presence and showcase my work, technical skills, and developer identity in a clear and professional way.
